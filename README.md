##Specifics and rules

Cod compiles with `clang++ -Wall -Wextra -Werror.`

I have to use only C++98 standard.

Except Module08, `STL algorithms` are forbidden.

Functions `alloc`, `free` and `printf` and keyword `friend` are forbidden too.
